# Google Search Console Setup for ngeoacademy.org

**Setup Date:** 2026-04-05
**Domain:** ngeoacademy.org
**Hosted On:** Vercel

---

## Setup Instructions

### Step 1: Access Google Search Console

1. Go to [search.google.com/search-console](https://search.google.com/search-console)
2. Sign in with the Google account that will manage the site
3. Click **"Add Property"**

### Step 2: Choose Verification Method

**Recommended: URL Prefix Method**

1. Select **"URL prefix"** (not Domain)
2. Enter: `https://ngeoacademy.org`
3. Click **Continue**

### Step 3: Verify Ownership (Vercel Method)

**Option A: HTML File Upload (Easiest)**

1. Download the HTML verification file from Google
2. Add the file to `/ngeoa-website/public/` folder
3. Deploy to Vercel
4. Click **Verify** in Search Console

**Option B: HTML Meta Tag**

1. Copy the meta tag Google provides, e.g.:
   ```html
   <meta name="google-site-verification" content="YOUR_CODE_HERE" />
   ```

2. Add to `/ngeoa-website/src/app/layout.tsx` in the metadata:
   ```typescript
   export const metadata: Metadata = {
     // ... existing metadata
     verification: {
       google: 'YOUR_CODE_HERE',
     },
   };
   ```

3. Deploy to Vercel
4. Click **Verify** in Search Console

**Option C: DNS TXT Record (Via Vercel)**

1. Copy the DNS TXT record Google provides
2. Go to Vercel Dashboard → ngeoa-website → Settings → Domains
3. Add DNS record: Type `TXT`, Value: Google's verification string
4. Wait 5-10 minutes for propagation
5. Click **Verify** in Search Console

### Step 4: Submit Sitemap

1. In Search Console left sidebar, click **Sitemaps**
2. Enter: `sitemap.xml`
3. Click **Submit**
4. Status should show "Success" after crawling

The sitemap at `https://ngeoacademy.org/sitemap.xml` includes:
- Homepage (priority 1.0)
- /program (priority 0.9)
- /apply (priority 0.9)
- /employers (priority 0.8)
- /about (priority 0.7)
- /contact (priority 0.7)

### Step 5: Request Indexing

1. In Search Console, use **URL Inspection** tool
2. Enter homepage URL: `https://ngeoacademy.org`
3. Click **Request Indexing**
4. Repeat for key pages (/program, /apply)

## Post-Setup Configuration

### Enable Email Alerts

1. Go to Settings (gear icon)
2. Enable email notifications for:
   - Coverage issues
   - Security issues
   - Manual actions

### Link Google Analytics (If Using)

1. In Google Analytics, go to Admin → Property Settings
2. Link Search Console property
3. This enables search query data in Analytics

### Monitor Weekly

Check these reports weekly:
- **Performance**: Search queries, clicks, impressions
- **Coverage**: Indexing status, errors
- **Mobile Usability**: Mobile-friendly issues
- **Core Web Vitals**: Page experience metrics
