## Daily rhythm (Mon–Fri)

| Time block | Action |
|------------|--------|
| Start of day | Open CRM/task list; note **top 3 outcomes** for the day. |
| Core | **Employer/partner outreach** — target **10 meaningful touches/day** (see definition below). |
| End of day | Log every touch in CRM; flag **follow-up date**; send **one-line** update to Cynthia if she owns a follow-up. |

**“Touch” definition (count toward 10):**

- Completed call (voicemail counts as 0.5 if logged with callback time).
- Email sent to a named contact with program one-pager attached.
- In-person visit scheduled or completed.
- Letter of support request sent (org + contact named).

**Not counted:** duplicate calls same day same person; internal team-only calls.

## CRM minimum fields (every employer / partner contact)

Log these for **each** organization:

| Field | Notes |
|-------|--------|
| `org_name` | Legal or DBA |
| `org_type` | Employer / association / funder / vendor / other |
| `contact_name` | Primary |
| `title` | |
| `phone` | |
| `email` | |
| `sector` | e.g. waste, aggregates, construction, logistics |
| `pipeline_stage` | See stages below |
| `last_touch_date` | |
| `next_action` | One line |
| `next_action_owner` | Brian / Cynthia / other |
| `next_action_due` | Date |
| `notes` | Brief, dated entries |

**Pipeline stages (employers):**

1. `identified` — on list, not contacted  
2. `contacted` — first touch  
3. `conversation` — real dialogue  
4. `committed` — verbal agreement to interviews/partnership (date who said what)  
5. `documented` — LOI/MOU/LOS signed or in legal review  
6. `paused` / `lost` — reason code
